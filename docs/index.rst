The ``quantum`` manual
===================================

Welcome to the ``quantum`` manual. ``quantum`` is a lightweight and WAN oriented software defined network (SDN), that is designed with security and scalability at its heart. This manual contains a full reference to ``quantum``, and indepth descriptions of the its inner workings.
